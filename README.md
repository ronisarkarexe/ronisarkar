# Roni Sarkar - Portfolio Website

A modern, responsive portfolio website showcasing my professional experience, skills, projects, education, and achievements. Built with Next.js 14 and featuring smooth animations, a clean UI, and an intuitive navigation system.

## üöÄ Features

- **Modern Design**: Clean, gradient-based UI with smooth animations powered by Framer Motion
- **Responsive Layout**: Fully responsive design that works seamlessly on all devices
- **Interactive Navigation**: Smooth scroll navigation with active section highlighting
- **Sections**:
  - Hero section with introduction and social links
  - Professional experience timeline
  - Skills showcase with interactive cloud visualization
  - Featured projects portfolio
  - Educational background
  - Honors and awards
  - Contact and hire section
- **Performance Optimized**: Built with Next.js 14 for optimal performance and SEO
- **Type-Safe**: Written in TypeScript for better code quality and developer experience

## üèÉ Getting Started

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd ronisarkar
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   ```

4. **Open your browser**
   Navigate to [https://ronisarkar.vercel.app](https://ronisarkar.vercel.app) to see the portfolio.



## üé® Customization

### Updating Content

- **Hero Section**: Edit `src/components/hero_view.tsx`
- **Experience**: Modify `src/components/experiences.view.tsx`
- **Projects**: Update `src/components/projects.view.tsx`
- **Skills**: Edit `src/components/skills.view.tsx`
- **Education**: Modify `src/components/education.view.tsx`
- **Awards**: Update `src/components/honors_award.view.tsx`
- **Contact**: Edit `src/components/contact_and_hire_me.view.tsx`

### Styling

The project uses Tailwind CSS for styling. You can customize:
- Colors: Edit `tailwind.config.ts`
- Global styles: Modify `src/app/globals.css`
- Component styles: Update individual component files

## üö¢ Deployment

### Deploy on Vercel (Recommended)

The easiest way to deploy this Next.js portfolio is using [Vercel](https://vercel.com/):

1. Push your code to GitHub
2. Import your repository on [Vercel](https://vercel.com/new)
3. Vercel will automatically detect Next.js and configure the build settings
4. Click deploy and your portfolio will be live!
